# FNK0107 quick notes

- Docs index: https://docs.freenove.com/projects/fnk0107/en/latest/index.html
- Components chapter: https://docs.freenove.com/projects/fnk0107/en/latest/fnk0107/codes/tutorial/1_Introduction_to_Main_Components_.html

Relevant for this NAS build:

- Built for Raspberry Pi 5.
- Supports 1/2/4 NVMe adapter variants.
- Quad board supports up to 4 NVMe SSD simultaneously.
- M.2 M-Key NVMe, sizes 2230/2242/2260/2280.
- Extra power feed and strong PSU are important for stability.
